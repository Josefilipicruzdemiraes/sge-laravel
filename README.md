# SGE – Sistema de Gerenciamento de Equipamentos

Aplicação web desenvolvida em Laravel, utilizando o padrão MVC e implementando
um sistema de gerenciamento de dados com operações CRUD
(Create, Read, Update e Delete).

## Objetivo
Permitir o cadastro, consulta, edição e exclusão de equipamentos e seus ensaios
técnicos, auxiliando no controle e organização das informações.

## Requisitos Funcionais
1. Cadastro de usuários no sistema
2. Autenticação de usuários (login/logout)
3. Cadastro de equipamentos
4. Listagem de equipamentos
5. Edição de equipamentos
6. Exclusão de equipamentos
7. Cadastro de ensaios técnicos
8. Associação de ensaios a equipamentos
9. Visualização de ensaios cadastrados
10. Dashboard com resumo de informações
11. Controle de acesso por usuário
12. Registro de data e responsável pelo ensaio

## Tecnologias Utilizadas
- PHP 8+
- Laravel
- MySQL
- Blade
- Bootstrap

## Estrutura do Projeto
- Models: regras de negócio
- Controllers: lógica do CRUD
- Views: interface com o usuário
- Routes: rotas do sistema
